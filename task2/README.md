# ArgoCD Challenge - part 2

- The [html file](argocd-flow.html), contains a small presentation about what happens with ArgoCD when a change hits the repo.

- To run it: open the file in your browser.

- Alternatively you can run an HTTP server to serve the HTML page:

```bash
python -m python3 -m http.server 8791
```

and then open [http://localhost:8791](http://localhost:8791)
